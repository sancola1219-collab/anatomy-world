# 寫實外部形態畫風規範（重畫基準）

目標：把「外部形態」視圖從粗糙色塊，提升到**生物課本插畫**等級。基準參考：人類外部形態、鱷魚外部形態（已達標）。

## 七項必備要素（每個物種外部形態都要有）
1. **正確比例與姿態**：符合該物種真實體型與典型姿勢（四足側面站姿、鳥站姿、魚側面、昆蟲背視…）。不要頭大身小的卡通比例。
2. **明暗立體**：主體用 `linearGradient`/`radialGradient` 做受光面亮、背光面暗。至少一個主漸層。
3. **表面質感**：依物種加紋理——毛髮走向短線、鱗片 `<pattern>`、羽毛層次、外骨骼分節、斑紋/花紋。用低透明度細線或 pattern，不要死板純色。
4. **五官/細節到位**：眼睛要有立體（漸層虹膜＋瞳孔＋一點高光）、該有的鼻/口/耳/觸角/鰭條等要畫出。
5. **輪廓平滑**：用貝茲曲線 `C`/`Q`，不要生硬折線拼湊。輪廓線比內部線稍粗（stroke-width 1.4–1.8）。
6. **柔和陰影/高光疊層**：可用半透明深色/淺色形狀做肌肉、身體圓潤感（opacity 0.15–0.4）。
7. **配色自然**：用該物種真實色調的漸層，避免螢光或死板單色。

## 硬性規則（不可破壞既有功能）
- **只重畫「外部形態」視圖的 `svg` 字串**（id 通常是 `external`）。其他視圖（骨骼/內臟/生命週期/循環…）**不要動**。
- 保留該視圖原有的 `parts[]` 陣列與所有 `data-part` 名稱：**每個 `parts[].name` 都要有對應的 `data-part`**，名稱完全一致（引擎點擊高亮靠這個）。可以新增細節，但不可刪掉原有標註部位。
- 標註層：引線＋文字包在 `<g class="labels">` 內，引線細灰線 `stroke="#666"`（或 #555）width 1，文字 `font-size 12–14 fill="var(--ink)"`。
- SVG 只寫內層標記，不要最外層 `<svg>`。
- 漸層/pattern 的 `id` 全檔唯一，**加物種前綴**（如 `dog-fur`、`frog-skin`），避免多物種同頁衝突。**注意：同一資料檔內既有其他視圖可能已用某些 id，務必用新前綴避免撞名。**
- `viewBox` 可調整以配合更好的構圖，但要與圖內容一致。
- 改完必須 `node --check <檔案>` 通過。

## 範例（鱷魚的做法，可參考）
- 主體 olive 綠漸層 + 鱗片 `<pattern>` 疊在上面
- 背部鋸齒骨板脊用一排小三角 path
- 眼睛：外圈深色隆起 + 黃色徑向漸層虹膜 + 垂直裂瞳 + 高光
- 牙齒：上下交錯的小三角
- 近側肢亮、遠側肢暗（前後景層次）
- 淺色腹部帶 + 橫鱗板細線

## 交付
- 逐一重畫指定物種的外部形態視圖。
- 回報：每個物種做了哪些提升、`data-part`↔`parts` 是否一致、`node --check` 結果。不要回傳整份程式碼。
