# 🔬 生物構造教學室 · Biology Anatomy Teaching Room

一個**離線可用**的互動生物構造教學網頁：依分類學呈現動物、植物、真菌的**寫實著色解剖圖**，支援**遠看整體、近看器官**，適合課堂教學與自學。純前端、零依賴、無需安裝。

> **45 物種 · 84 視圖 · 517 個構造標註**，涵蓋動物界、植物界、真菌界。

## ✨ 特色

- 🌳 **分類樹導覽**：界 → 類群 → 物種，可折疊、可搜尋
- 🧊 **3D 立體模型**：可旋轉、縮放、分層拆解（外觀／骨骼／器官）的立體檢視（Three.js，本機化維持離線）
- ▶️ **生理過程動畫**：心臟跳動、血液脈動、肺／鰓呼吸、翅鰭擺動；生命週期／變態／授粉／世代交替**逐階段循環播放**（可一鍵開關）
- 🔍 **遠看／近看**：滾輪縮放＋拖曳平移，近距離觀察器官細節
- 🧬 **逐系統拆解**：分頁切換外部形態／骨骼／內臟／橫切面／生命週期…
- 🎨 **寫實著色 SVG**：漸層明暗、教科書式解剖標註，可縮放不失真
- 🏷️ **互動標註**：點器官高亮、標註可開關、跨物種搜尋器官（如搜「心臟」）
- 🌓 深淺色主題，響應式版面

## 🚀 使用方式

直接用瀏覽器打開 `index.html` 即可，無需伺服器。

若瀏覽器限制本地載入，可用任一靜態伺服器：

```bash
python -m http.server 8000
# 然後開 http://localhost:8000
```

## 📂 專案結構

```
├─ index.html                 進入點
├─ css/style.css              樣式與深淺色主題
├─ js/app.js                  核心引擎（通用，不寫死物種）
├─ data/
│  ├─ SCHEMA.md               物種資料規格
│  └─ *.js                    各分類的解剖 SVG 內容
├─ verify.js                  一鍵驗收工具（node verify.js）
├─ lesson.md                  設計決策與修正筆記
└─ HANDOFF.md                 完整接手指南
```

## 🛠️ 擴充與貢獻

新增物種只需在對應 `data/*.js` 複製一個 `ANATOMY.register({...})` 區塊。規格見 [`data/SCHEMA.md`](data/SCHEMA.md)，完整開發流程與驗證見 [`HANDOFF.md`](HANDOFF.md)。

改完後執行三層驗證：

```bash
node --check data/*.js   # 語法
node verify.js           # 整合（應 0 錯誤）
# 瀏覽器互動檢查
```
